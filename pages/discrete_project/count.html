Climate Change

Why is it so Difficult to Tackle?

Climate Change is so difficult to tackle for many reasons, but one reason is that there are so many competing view points on how to address it. These competing approches are often mutually exclusive, limited by the financial resources a state, corporation, or individual has.

Within these four approaches there are competing views as well. As we can see with clean energy, there are multiple ways the goal of cleaner energy can be reached. This means there are countless ways these solutions can be selected.

For example, California wants to invest in cleaner energy and improve transportation systems. However, they only have enough money to fund four energy projects and three transportation projects. If there are 14 energy proposals and 12 transportation proposals, California has 220,220 different combinations of proposals to fund.

We can find the number of combinations using the binomial coeeficient. n is the number of proposals and k is the number that we can fund.

( n k ) = n ! k ! ( n − k ) !

n! represents the number of ways to fund all proposals but the order in which we fund them is relevant. There are n options for the first proposal to fund, then there are n-1 options for the next proposal to fund and so forth. The number of options multiplied together represent the total ways to arrange because for each proposla we fund the number possible arrangements increases one for each option we have. We divide n! by (n-k)! hecasue we want to remove all the arrangements that select more than k proposals. This removes all but the first k terms from n!. It doesn't matter in what order we select the proposals to fund so next we divide by k! to remove the counts where the projects funded are the same, but selected in a different order. k! is how many ways we can arrange the k proposlas we selected.

Applying this to our hypothetical, we have 14 energy proposals and must choose 4.

( 14 4 ) = 14 ! 4 ! ( 14 − 4 ) ! = 1001

Repeat this with our 12 transportation proposals.

( 12 3 ) = 12 ! 3 ! ( 12 − 3 ) ! = 220

Each possible combination of energy porjects can be combined with each possible combination of transportation projects so we multiply them.

1001 * 220 = 220,220
